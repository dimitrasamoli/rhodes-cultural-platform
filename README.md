# Rhodes Cultural Platform 

A multilingual (GR/EN) web platform designed to promote and visualize the cultural activities of Rhodes.

The project combines web development and open data concepts to create a sustainable digital space for local cultural events.

Built with **React**, **Tailwind CSS**, **Leaflet (OpenStreetMap)**, and a **Python data pipeline** that automatically converts Google Form submissions into structured JSON data.

## Data Pipeline
The platform includes an automated pipeline for updating cultural events. 
```bash
Google Form ‚Üí CSV (sheet.csv) ‚Üí csv_to_json.py ‚Üí src/data/events.json ‚Üí React pages
```

### Data Processing 
1. Cultural organizers submit event data through a **Google Form**
   
3. Google Sheet exports responses as a **CSV file** (sheet.csv)
   
5. The Python script csv_to_json.py:
   - loads the CSV using **Pandas**
   - cleans & tranforms the data
   - generates a structured JSON file:
   ```bash
   src/data/events.json

6. React imports this JSON directly
   ```bash
   import eventsData from "../data/events.json";
   
7. All event-related pages update automatically during the build

## Regenerating Event Data 
Whenever new Google Form responses are added: 
1. Download the updated CSV from Google Sheets
   ```bash
   ‚Üí File ‚Üí Download ‚Üí CSV
   
2. Place it in:
   ```bash
   scripts/sheet.csv
   
3. Run the conversion script:
   ```bash
   cd scripts
   python csv_to_json.py
   
4. A fresh src/data/events.json is generated
   
6. Restart the dev server or rebuild the app

## Features 
- **Dynamic Activities** - events automatically loaded form JSON data generated by a Python script
- **Interactive Map** - cultural events visualized with Leaflet and OpenStreetMap
- **Multilingual Interface** - full bilingual support (Greek & English) using i18next
- **Plan Your Trip** - personalized itineraty overview
- **Experiences Section** - highlights of local traditions and workshops
- **Open-Source & Sustainable** - free of proprietary APIs or billing dependencies
